## 1. 랜딩페이지 카피

### 한 줄 소개

**BET — PT의 "보이지 않는 성장"을 눈에 보이게 만드는 AI 트레이너**

### 서브 카피

- 인바디 수치 한 장으로 끝나던 회원 성과를, 다차원 그래프로 잠재회원에게 증명하세요.
- 회원의 PT 외 6일 24시간을 AI 트레이너가 관리합니다. 생활습관 데이터가 쌓일수록 트레이너의 피드백은 더 날카로워집니다.
- 헬스장 단위 월 구독. 회원 1인당 월 9,900원부터. (헬스장이 패키지화해서 회원에게 전가하는 것도 가능)

### 핵심 한 줄

> "PT 품질의 증거를, 트레이너가 아닌 데이터가 말하게 합니다."

### 랜딩 CTA

- "[무료] 3개월 파일럿 신청" → 12주 간 3개 회원 계정 무료 제공, 이후 성과 확인 후 정식 계약.

## 2. 해결하려는 문제 (Pain Statement)

### 트레이너·관장 관점

1. **"우리 PT 좋아요"를 증명할 자료가 빈약함.** 상담 테이블에서 인바디 수치 하나만 올려놓고 설득해야 하는데, 인바디는 회원의 식단·수면에 크게 흔들리는 표면 지표라 PT 품질 증명력이 낮음.
2. **회원 발전 추이 가시화가 기존 도구에서 제공되지 않음.** Rappo 등 공유 캘린더·채팅 일지 도구에는 이 기능이 없어, 회원이 실제로 얼마나 변했는지를 누구도 눈으로 확인하지 못함.
3. **PT 외 6일을 방치**. 주 1회 PT만으로 성과가 결정되지 않음을 트레이너 본인도 알지만, PT 외 시간 회원 습관을 직접 관리할 리소스가 없음.
4. **리텐션 고민**. 회원이 "3개월 하고 빠지는 패턴"을 끊고 싶지만, 지속적 동기부여 장치가 부재.

### 회원(downstream) 관점

1. PT 받는 날만 운동에 집중하고 나머지 6일은 무방비 상태.
2. 내가 발전하고 있는지가 애매 — 인바디 외에는 피드백이 산발적 채팅 일지에 묻혀있음.
3. 트레이너 퇴직·이직 시 축적된 내 운동 히스토리가 증발.

## 3. BET의 해결책 (How)

### 핵심 3개 축

#### (A) 회원 성장 그래프 대시보드

- **자동 수집 지표**: PT 세션별 중량/볼륨 진전, 자세 교정 항목 추이, 회원 자가보고 (수면/식단/컨디션) 누적, 인바디 연동.
- **트레이너 뷰**: 회원별 대시보드에서 12주 진전, 정체, 부상 경고등을 한 화면에서 확인.
- **관장 뷰 (상담용)**: "이 트레이너가 맡은 회원 N명의 평균 발전 곡선"을 잠재회원에게 바로 제시 가능. 기존 인바디 자료를 대체/보완.

#### (B) AI 트레이너 (PT 외 시간)

- 회원이 카카오톡 채널 또는 BET 앱에서 AI 트레이너와 대화. 식단 사진/운동 자가영상/컨디션 보고를 받아 당일 루틴 조정.
- AI는 해당 회원의 PT 기록·목표·트레이너의 프로토콜을 컨텍스트로 씀. 즉 "우리 트레이너의 색깔을 이해하는 AI".
- 위험 신호 (과훈련, 급격한 체중 변화, 부상 징후) 감지 시 트레이너에게 알림.

#### (C) Rappo 병행 전제 운영 도구

- **캘린더/일정 기능은 만들지 않음.** Rappo가 잘 하는 영역은 그대로 두고, BET은 Rappo API 또는 CSV import 로 회원 목록만 동기화. 트레이너의 이중 입력 부담 제거.
- 회원 등록 시 Rappo 연락처를 기반으로 자동 초대 링크 발송.

### 트레이너가 하게 되는 것 / 하지 않게 되는 것

| 항목              | Before (현재)     | After (BET 도입)                         |
| ----------------- | ----------------- | ---------------------------------------- |
| PT 외 시간 코칭   | 단발 카톡/없음    | AI가 대행, 이상 징후만 트레이너에게 에스컬레이션 |
| 회원 성과 자료    | 인바디 한 장      | 다차원 성장 그래프, PDF 리포트 출력 가능 |
| 회원 리텐션 대화  | 막연한 격려       | 데이터 기반 "당신 지난 4주 동안 이 지점 개선됨" |
| 과거 기록 열람    | 채팅 스크롤       | 대시보드 검색                            |

## 4. 가격 플랜 (세일즈 미팅 단계에서만 공개)

- **파일럿 (12주)**: 무료. 회원 3명까지. 리포트 기능 전체 개방. 정식 계약 전환 시 조건 없음.
- **스탠다드**: **회원 1인당 월 9,900원**. 최소 10명부터.
- **엔터프라이즈** (50명 이상): 월 인당 8,000원.
- **연 일시불 결제 시 10% 할인.**
- 헬스장이 회원에게 요금을 어떻게 전가할지(패키지 판매, 무료 제공)는 자율. BET은 헬스장에만 청구.

참고: 회원당 월 1만 원 선은 업계 인터뷰상 지불 임계점으로 확인됨. 파일럿을 통해 "수치화된 회원 발전 증거" 를 먼저 체감시킨 뒤 유료 전환하는 구조.

## 5. 자주 받는 질문 (세일즈 미팅용 FAQ)

### Q1. Rappo를 해지해야 하나요?

아니요. BET은 Rappo의 캘린더·운영관리 기능을 대체하지 않습니다. 회원 동기화만 하고 나머지는 Rappo를 그대로 쓰셔도 됩니다. 목표는 "BET이 Rappo의 사각지대(성과 가시화 + PT 외 시간)만 채우는 것".

### Q2. 회원이 앱 설치·매일 기록을 귀찮아하면요?

AI 트레이너는 카카오톡 채널로도 동작하므로 앱 설치는 필수 아닙니다. 입력 부담은 하루 1~2회 가벼운 자가보고 + 선택적 사진/영상. 회원이 아무 입력을 안 해도 PT 세션 기록만으로 대시보드는 채워집니다.

### Q3. 소속 트레이너들이 "내 방식대로 못 한다"며 반대하면요?

트레이너별로 프로토콜(운동 설계 규칙, 자세 교정 중점, AI의 어조·강도)을 커스터마이즈 가능. AI가 자기 색깔을 흉내내준다는 느낌을 강조.

### Q4. 데이터가 우리 회원 데이터인가, BET 데이터인가?

헬스장 소유. 계약 해지 시 전체 CSV/PDF 내보내기 제공. BET은 익명화된 집계 데이터만 제품 개선에 활용.

### Q5. 트레이너가 퇴사하면 쌓인 AI 컨텍스트는?

회원 단위로 귀속. 트레이너 교체 시 새 트레이너가 해당 회원의 AI 컨텍스트를 그대로 승계.

- 기간: 12주 (3개월).
- 대상 회원: 관장 본인이 맡는 회원 3명 (소속 트레이너 개입 최소화).
- 무료. 3개월 종료 시점에 다음 2개 자료를 드림:
  1. 해당 3명의 "12주 성장 리포트" PDF. 이걸 그대로 다음 상담에 쓸 수 있도록.
  2. 리텐션·재등록 의사 변화 비교.
- 성과가 미진하면 정식 계약은 없음. 데이터만 남음.

## 7. BET이 약속하지 않는 것 (리스크 솔직 고지)

- 인바디 수치의 **단기** 개선을 약속하지 않음. AI가 하는 건 "생활습관 일관성"이지 "빠른 결과"가 아님.
- 트레이너의 자세 교정을 대체하지 않음. BET은 그 시간을 확보해 주는 도구.
- 헬스장 매출을 직접 올리는 마케팅 도구 아님. "상담 전환율"을 올릴 자료를 제공하는 도구.
