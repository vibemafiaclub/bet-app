# pt-trainer-owner-01 — 미확인 정보 체크리스트

페르소나 `profile.md`에서 `unknown`으로 남겨둔 것, 또는 가정값으로 채운 것 중
다음 인터뷰/관찰로 확정 가능한 항목들. 확인되면 `profile.md` 업데이트 후
`version`을 +1, `updated_at` 갱신.

## 정량 정보 (다음 PT 때 확인)

- [ ] 실제 PT 회원 수 (현재 가정: 50명)
- [ ] Rappo 월 지출액 (switching_cost 정교화용)
- [ ] 최근 6개월 회원 리텐션율 / 평균 PT 지속 기간 (문제의 실제 크기 검증)
- [ ] 신규 회원 유치의 주요 채널 (지인 추천 / 네이버 플레이스 / 인스타 / 전단 / …)
- [ ] 가치 증명 시 지불 상한 (현재 "인당 월 ~1만원"은 가치 미검증 상태 추정치)

## 조직·도구

- [ ] Rappo 외 운영·회계·CRM 도구 사용 여부 (현재 추정: 없음, Rappo가 운영관리 겸용)
- [ ] 소속 트레이너 인원, 각자 회원 풀 독립성
- [ ] 소속 트레이너의 `trust_with_keyman` 초깃값
- [ ] 소속 트레이너의 `tech_literacy`, 새 도구 도입에 대한 일반적 태도

## 의사결정 맥락

- [ ] 과거 도구 도입 실패 경험 (reject_triggers 정교화)
- [ ] Rappo 이전에 시도했다 접은 도구 있는지
- [ ] 헬스장 외 재무 의사결정 영향자 존재 여부 (공동운영자 / 배우자 등)
- [ ] 헬스장 월 고정비 구조 (임대료, 인건비 비중) — 회원당 월 1만원의 상대적 체감 파악

## downstream 검증 (회원 측)

- [ ] 현재 회원들의 연령대 분포 (생활습관 입력 UX 부담 체감도에 영향)
- [ ] 회원이 Rappo 채팅 일지를 실제로 쓰는지, 쓴다면 빈도
- [ ] 회원이 PT 외 시간 코칭에 대해 표한 니즈 유무
